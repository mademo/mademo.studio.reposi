Conçois un site expérimental et éditorial pour Mademo, artiste, designer, photographe et autrice.

Le site ne doit pas être un portfolio traditionnel composé uniquement de projets terminés. Il doit fonctionner comme un atelier vivant, un carnet de recherche public et une archive artistique évolutive.

L’objectif principal est de permettre à l’artiste de documenter facilement :

ses projets en cours ;
ses recherches ;
ses expérimentations ;
ses références ;
ses photographies ;
ses textes ;
ses essais ;
ses questions ;
ses échecs ;
les liens entre ses différentes pratiques.
Concept général

Créer une expérience appelée :

L’atelier vivant

Le visiteur doit pouvoir observer non seulement les résultats, mais aussi la manière dont les idées apparaissent, évoluent, se croisent et se transforment.

Le site doit donner l’impression d’entrer dans une pensée en mouvement.

Interaction innovante

Créer une barre latérale appelée :

Fil de recherche

Lorsqu’un visiteur consulte plusieurs fragments, il peut les ajouter à son fil.

Le site construit alors un parcours temporaire montrant les liens entre les éléments sélectionnés.

Exemple :

Photographie → note sur le corps → expérimentation lumineuse → texte sur la perception → projet L’Œil de la matière.

Le visiteur peut :

conserver ce parcours pendant sa visite ;
revenir aux éléments précédents ;
afficher les connexions ;
générer une page de synthèse du parcours.

Cette interaction doit être subtile et simple à comprendre.
